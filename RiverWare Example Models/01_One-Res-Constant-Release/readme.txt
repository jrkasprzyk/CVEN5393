The models in this folder are simple examples that try to meet a constant demand in each period.

example-usgs-dmi.mdl demonstrates the use of a USGS DMI to get input data

one-res-constant-release.mdl demonstrates a simple ruleset that delivers water and avoids overtopping - only refer to this model if you need an application that has less sophisticated rules than the spring 2026 version below

one-res-constant-release-spring2026.mdl an updated model that has more logic and better naming conventions
